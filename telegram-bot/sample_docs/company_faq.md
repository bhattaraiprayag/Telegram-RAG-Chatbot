## Security and Compliance

### Is my data secure?

Absolutely. Security is our top priority:

- All data is encrypted at rest (AES-256) and in transit (TLS 1.3)
- We undergo annual SOC 2 Type II audits
- GDPR and CCPA compliant
- ISO 27001 certified
- Regular penetration testing by third-party security firms

### Where is data stored?

Data is stored in secure cloud infrastructure:

- **US customers**: AWS US-East and US-West regions
- **EU customers**: AWS Frankfurt data center (GDPR compliant)
- **APAC customers**: AWS Singapore and Tokyo regions

### Can I export my data?

Yes, you can export all your data at any time in standard formats (CSV, JSON, XML). Go to Settings > Data Management > Export to initiate a data export.
